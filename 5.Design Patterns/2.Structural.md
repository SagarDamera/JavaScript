# Structural Patterns

- Module Pattern
- Facade Pattern
- Adapter Pattern
- Decorator Pattern

## 2.1 Module Pattern

Module Pattern is used to organize code and keep related variables/functions together.

Keeping calculator methods like add and subtract inside one calculator module.

```js
const Calculator = (() => {
  function add(a, b) {
    return a + b;
  }

  function subtract(a, b) {
    return a - b;
  }

  return {
    add,
    subtract,
  };
})();

console.log(Calculator.add(5, 3));
console.log(Calculator.subtract(5, 3));
```

### Is Module Pattern used in React?

Yes. React components are usually written as modules and exported/imported between files.

```jsx
function Header() {
  return <h2>Welcome to My App</h2>;
}

export default Header;
```

```jsx
import Header from "./Header";

function App() {
  return <Header />;
}

export default App;
```

## 2.2 Facade Pattern

Facade Pattern provides a simple interface to complex code.

A remote control hides complex TV operations behind simple buttons like power, volume, and channel.

```js
class VideoPlayer {
  loadVideo() {
    console.log("Loading video");
  }

  playVideo() {
    console.log("Playing video");
  }

  loadSubtitles() {
    console.log("Loading subtitles");
  }
}

class VideoPlayerFacade {
  constructor() {
    this.player = new VideoPlayer();
  }

  playMovie() {
    this.player.loadVideo();
    this.player.loadSubtitles();
    this.player.playVideo();
  }
}

const movie = new VideoPlayerFacade();

movie.playMovie();
```

### Is Facade Pattern used in React?

Yes. We can create a simple component that hides complex UI logic inside it.

```jsx
function UserCard({ user }) {
  return (
    <div>
      <h2>{user.name}</h2>
      <p>{user.role}</p>
      <p>{user.email}</p>
    </div>
  );
}

function App() {
  const user = {
    name: "Sagar",
    role: "Developer",
    email: "Sagar@example.com",
  };

  return <UserCard user={user} />;
}

export default App;
```

## 2.3 Adapter Pattern

Adapter Pattern is used to make incompatible data or code work together.

A phone charger adapter helps connect a charger plug to a different socket type.

```js
const oldUserData = {
  first_name: "Sagar",
  job_title: "Developer",
};

function userAdapter(user) {
  return {
    name: user.first_name,
    role: user.job_title,
  };
}

const newUserData = userAdapter(oldUserData);

console.log(newUserData.name);
console.log(newUserData.role);
```

### Is Adapter Pattern used in React?

Yes. It is often used to convert API response data into the format needed by React components.

```jsx
function userAdapter(apiUser) {
  return {
    name: apiUser.first_name,
    role: apiUser.job_title,
  };
}

function UserProfile({ user }) {
  return (
    <div>
      <h2>{user.name}</h2>
      <p>{user.role}</p>
    </div>
  );
}

function App() {
  const apiUser = {
    first_name: "Sagar",
    job_title: "Developer",
  };

  const user = userAdapter(apiUser);

  return <UserProfile user={user} />;
}

export default App;
```

## 2.4 Decorator Pattern

Decorator Pattern is used to add extra behavior to an object or function without changing the original code.

Adding extra toppings to a basic pizza.

```js
function basicCoffee() {
  return "Coffee";
}

function addMilk(coffee) {
  return coffee() + " + Milk";
}

function addSugar(coffee) {
  return coffee() + " + Sugar";
}

const coffeeWithMilkAndSugar = addSugar(() => addMilk(basicCoffee));

console.log(coffeeWithMilkAndSugar());
```

### Is Decorator Pattern used in React?

Yes. Higher Order Components are a common example of Decorator Pattern in React.

```jsx
function UserProfile({ name }) {
  return <h2>{name}</h2>;
}

function withBorder(Component) {
  return function DecoratedComponent(props) {
    return (
      <div style={{ border: "1px solid black", padding: "10px" }}>
        <Component {...props} />
      </div>
    );
  };
}

const UserProfileWithBorder = withBorder(UserProfile);

function App() {
  return <UserProfileWithBorder name="Sagar" />;
}

export default App;
```
